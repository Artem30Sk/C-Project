Sweetshop - курсовой проект по теме: "Настольное приложение для администратора кондитерского магазина"

Для правильного запуска проекта необходимо:

1. Иметь установленные:
 Microsoft Visual Studio 2019
 Microsoft SQL Server Management Studio v18

2. Используемые проекты:
Sweetshop - проект WPF приложения
DBase - скрипт базы данных

После скачивания всех файлов, необходимо проверить строку подключения с БД.
